# **Alltra - Network Traffic Manager**

## **Description**

Alltra is an example application for network traffic management. It includes features such as animations, login, sign-up, and dashboard functionalities. The app demonstrates how to handle data flow and user interaction effectively.

---

## **Features**

1. Login and sign-up functionality.
2. Real-time dashboard to display network metrics.
3. Smooth animations for enhanced user experience.
4. Basic data traffic tracking features.
5. Modular design for future expansion and scalability.
